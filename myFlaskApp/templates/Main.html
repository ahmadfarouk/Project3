 Summary: Netflix Data Analysis 

 Visualization 

    • The purpose of this project is to analyze Netflix data.The foundation of this project is Project-2 which was normalizing the Netflix data and import them in PorsgrSQL.The purpose of this project is to provide users an interactive means to explore Netflix data. To accomplish this analysis, we first pulled data from Azure PostgreSQL to assemble a dataset on over 3000 records. After assembling the dataset, we used Flask to publish data in to index.html .

    Factors we looked at include: Country, Rating, Revenue, and Budget. This site provides the visualizations created as part of the analysis which are deployed via js, d3 and plotly/Leaflet.
